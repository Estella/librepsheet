About librepsheet

librepsheet was built to provide a common abstraction layer for the Repsheet logic. Since Repsheet targets more than one web server and shares functionality with the backend this library provides one commmon place for business logic. This allows for implementations of web and backend Repsheet modules to focus on just the platform specific details.
